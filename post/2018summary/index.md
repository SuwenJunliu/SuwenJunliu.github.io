# Last semester

## 集中式学习
开学之后，由于我们只有一门英语课，所以根本不用担心课程的问题，再加上很多同学都选择去校外做毕业设计，因此只有大概一半的学生还在学校继续做毕设。甚至于，有同学直接不来上课了，用这个时间把驾照考完。我只能说：~~实在是太机智了~~

开学之后我依然在玩游戏或者是东逛西逛，按时上课。虽然听起来有些搞笑，但是我还是觉得不上课就有点难受。

毕业设计是真的坑，包括我的大创。真的是只有自己知道自己做的是有多么的糟糕。那些程序还等着我进行~~高雅~~**重构**。这又是不少的工作量。虽然做的很糟糕，但是由于自己Presentation还可以，~~也是我为数不多的强项之一~~，因此拿到了一个还可以的分数，混到了成绩。实际上自己面对这个成绩还是相当羞愧的。

大创也是一样，紧赶慢赶最后还是在结束之前勉强上线。想到大一下学期那时候的雄心壮志，真的是无语凝噎。我们单纯的认为，找一个比较年轻的老师就会有时间带我们，现在想想还是太天真了...
## 毕业前夕
对于2018年而言，最重要的时间节点当然是本科毕业了。

每当想到我的本科学校，我的心情是很复杂的。在一年级的很长一段时间内，我一直认为我是从一个“大学一样的高中”到了一所“高中一样的大学”。这样的想法困扰了我很长时间，一直到二年级才逐渐将生活的中心从不断的自怨自艾中抽离出来。

在毕业的前夕，我们一起去玩日租房。那天大家都在热热闹闹的做饭喝酒，我就坐在客厅用Xbox玩古墓丽影9，楼上舍友在开黑，一晚上吵吵闹闹的。直至现在提笔，我都觉得很多事情并没有想象的那么遥远。

# 雁栖湖生活
## 开学
很快，9月份就开学了。

和在吉林大学不一样，至少雁栖湖离家更近一些。只需要短短一天的时间就能走个单程。因为贫穷的原因，我只买了一张坐票2333，~~实际上是因为没有提前买票所以买不到票了~~。原因很简单，一开始以为还要去研究所，就拖着没买票，这一拖就拖到了开学。

来到北京之后，我第一个感觉就是累。当我带着大包小包的东西下车的时候已经是下午的四点多了。在招生群里看到的消息，校车在玉泉路校区并且很快就要发车。这下肯定是没办法按时到那儿坐校车了，无奈只能自己想办法出发。

打开手机发现，雁栖湖校区在60多公里之外的地方，还得倒腾城际公交车才能到达，只能是带着我的行李箱还有背包去东直门换乘。北京和南宁的公交车很不一样，都是分段计价。当我公交车票价时，我还在疑惑什么公交车是13块一次单程的，但是它开上高速公路的时候，我才意识到

夜幕来临之时，公交车依然行驶在高速公路上。我打开手机，很焦虑的在刷新自己的位置，看自己能否赶上在明珠广场换乘的班车。我记得不知在哪儿看到的一个说法，焦虑是来源于对未来的不确定。我不清楚我能否赶上换乘的H58，也不知道未来五年我能不能毕业，实现我的理想。

当车子摇摇晃晃到明珠广场时，已经是晚上的七点多，最后一班车是8：20(~~如果我没记错的话~~)。这时候很幸运的碰到了一起去国科大的小姐姐，交谈一番之后得知她是化学所的硕士生。尽管如此，我们还是没有交换联系方式。现在想来有些可惜，毕竟是在国科大遇到的第一个同学。

## 开学综合征
凡是新学期开学，我都会患上**开学综合征**

1. 选很多的课试图约束自己
2. 过高估计自己的能力
3. 生活不规律
4. 试图参加很多的活动丰富自己的经历

很不巧，上边都中了。稍显幸运的是，我刚开学的时候还没浪过头，还记得多看看英语。这边的英语是标准化测试，也就意味着只要努力，还是能穷尽题库的。于是我就在网上找了很多**北京市硕士学术英语**这类的东西，狠狠地刷了一把题。基本上将08~15年的题目都做了一遍。事实证明这还是有点效果的，阅读有一篇完全一样的题目。测试的听力就依靠大学仅剩的那点资本，勉强撑了过去。

有意思的是，这个免修考试是我第一次意识到这个学校不靠谱。大概在两天之后，免修考试的成绩就出来了。我查询的时候很惊喜的发现了自己通过了考试。没等高兴太久，突然发现自己的名字又没了。不止是我一个人，很多人都遇到了这样的问题。当时我心里就想，如果我能通过英语免修，我就多选课，如果不能，就选少一些。事实证明，我完美的把自己坑了。

但无论如何，我都成功完成了这个学期的学习任务，而且除了最后一个月之外，整体来看还是相对比较上进的。

## 课程学习
整体而言，最喜欢的课程果然还是**地球动力学**还有**高等地震学**。虽然陈老师让我随便选课，我还是选了我觉得比较重要的主干课程。在雁栖湖有P-10的辅修计划。本来我想学计算机的辅修，结果盘算了一下，自己无论如何都没办法修读**高等地震学**的同时把两门必修课都给选上，只能忍痛放弃了计算机辅修的计划。

如果放弃了计算机辅修，课程表一下子就空了很多。于是就将自己喜欢的课程都塞了进去。整个学期，我觉得比较可惜的课程大概就是**高等构造地质学**，整个课程是以讨论课的形式展开的，可是我之前本科学习的效果很不好，因此忙活了很长时间也没有学到有用的东西。另外就是**偏微分方程**，这门课内容很多，而且大多数都是点到为止。前期还好，后期作业基本靠抄，考试全看助教给的情面。如果能好好吸收课上所学还行，可惜课后也没有多少时间复习，因此非常的遗憾。

另外比较可惜的课程可能是英语b。虽然我很努力的学习了，最后的结果也不是很好。另外就是，外教对于抄袭之类的事情真的是零容忍。他们可以忍受你是个菜鸡，一直很菜很菜，也绝对不会希望你抄袭。学术写作有一个小哥从网上扒拉了一片essay交上去，在群里好一顿批斗，也不知道这个小哥最后成绩如何。估计按照外教的脾气，这课怕是要被Fail掉。
